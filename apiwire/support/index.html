Support

Use the right route for the right issue.

Public bugs and feature requests belong in GitHub Issues. Private support, security, privacy, and licensing each have their own route so the right details reach the right place.

Public bugs and ideas

Use GitHub Issues for functional defects, regressions, workflow problems, and feature requests. Public issue tracking opens closer to release.

Open GitHub Issues ↗

Private support

For direct help that should not live in a public thread, email support@aankda.ai.

support@aankda.ai ↗

Security issues

Do not file publicly

Report vulnerabilities privately to dev@aankda.ai with a subject such as:

[security] API Wire

Licensing and commercial use

Commercial licensing goes to license@aankda.ai. Legal and trademark matters go to legal@aankda.ai. Privacy questions go to privacy@aankda.ai.
