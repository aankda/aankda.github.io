Privacy Policy

Updated April 6, 2026

This policy covers both the public AuraEstate site at aankda.ai/auraestate and the AuraEstate browser extension. AuraEstate was built with privacy as a first principle. Here is exactly what runs locally, what reaches public map providers, and what never goes through an AANKDA backend.

Site

Static site. No AANKDA analytics.

The public AuraEstate site is a static public website. AANKDA does not run analytics, ads, chat widgets, or session-replay tools on the site.

GitHub, your browser, your ISP, and similar infrastructure may still process standard request metadata such as IP address, user agent, request time, and referrer in order to deliver the page.

Extension

Core processing stays in your browser.

AuraEstate computes numerology values, injects listing badges, renders the detail panel, and keeps filter state locally in your browser. The extension does not require an account, a subscription, or an AANKDA-run backend.

Network

Public map requests happen only when direction data is needed.

  • Providers: Photon, OpenStreetMap Nominatim, and Overpass API mirrors.
  • Sent only when needed: approximate coordinates or a first-line address string.
  • Requests include an X-Application header with the extension version and contact email for API compliance.
  • Providers and network infrastructure may still observe standard connection data such as IP address, user agent, request time, and request path.

Storage

Preferences and direction cache stay on-device.

  • Browser storage is used for preferences and active filter state.
  • IndexedDB is used to cache direction results locally for up to 30 days.
  • AANKDA does not sync this data to a remote service.

Permissions

The requested surface stays small.

  • Browser permission: storage.
  • Supported hosts: Zillow, Redfin, Trulia, Homes.com, Movoto, and Apartments.com.
  • Outbound links from the site go only to browser-store listings and the privacy email.

Contact

AANKDA LLC is the operator.

If you have privacy questions, email privacy@aankda.ai. If this policy changes materially, the date at the top of this page will be updated when the revised version is published.
