AuraEstate

Chaldean numerology and entrance-direction insight, overlaid directly on supported listings. Works on Zillow, Redfin, Trulia, Homes.com, Movoto, Apartments.com, and MLSListings, with public map lookups only when needed.

Also available in Firefox

What it does

◌

Chaldean Insight

Single-digit property signatures, visible directly on listing cards and detail pages.

⌖

Orientation Compass

Entrance-direction context estimated from public map geometry only when an address needs it.

◍

Privacy First

No accounts, no analytics, and no developer-run backend for numerology or filters.

How it works

01

Use your browser’s store and keep browsing normally.

Read the overlay

Numerology badges and direction context appear in place.

04

Privacy by design

Your search stays yours.

AuraEstate runs locally in your browser. When direction data is missing, it can make a minimal public map request using approximate coordinates or a first-line address, then cache the result on-device.

  • No analytics
  • No user accounts
  • Storage permission only
  • Public map requests only when needed
